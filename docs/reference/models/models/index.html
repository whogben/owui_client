    ModelParams

    Bases: BaseModel

    Model parameters configuration. Allows extra fields to store various model-specific parameters.

    Common parameters include: - system: System prompt string. - stop: List of stop sequences.

    ModelMeta

    Bases: BaseModel

    Metadata for a model.

    Additional fields often used (via extra="allow"): - knowledge (list): List of knowledge collection items. - toolIds (list[str]): List of tool IDs enabled for this model. - filterIds (list[str]): List of filter IDs enabled. - defaultFilterIds (list[str]): List of default filter IDs. - actionIds (list[str]): List of action IDs. - defaultFeatureIds (list[str]): List of default feature IDs. - suggestion_prompts (list): List of suggestion prompts. - tags (list): List of tags associated with the model.

    Access control settings for the model.

    Structure: - None: Public access, available to all users with the "user" role. - {}: Private access, restricted exclusively to the owner. - Custom permissions:
